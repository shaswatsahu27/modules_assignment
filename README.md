# 📘 Assignment 5: Modules & Packages in Python

## 📌 Problem Statement
This assignment focuses on organizing Python code into **modules and packages** to improve code reusability and structure. It demonstrates how to create custom modules, import them, and group related functionality into packages.

---

## 🎯 Objectives
- Create and use Python modules (`.py` files)
- Import modules using different methods
- Create and use packages with `__init__.py`
- Organize code in a structured and scalable way
- Practice clean and beginner-friendly coding
